# SaveChat Bot

Professional Telegram bot for message archiving and deleted message tracking.

## Features

- ✅ Auto-save all messages from monitored chats
- 🗑 Track deleted/edited messages
- 🔍 Full-text search through message history
- 👤 User statistics and analytics
- 🔐 Admin panel with database access
- ⚡ Rate limiting and security measures
- 🐳 Docker support
- 📊 Redis caching

## Tech Stack

- Python 3.11+
- aiogram 3.4+
- PostgreSQL 15
- Redis 7
- Docker & Docker Compose
